HTML 文本格式化元素
b标签定义粗体文本，该标签可以指定id，style,class等核心属性，还可以定义onclick等事件属性

i标签定义斜体文本，该标签可以指定id，style,class等核心属性，还可以定义onclick等事件属性

em标签定义强调文本，效果和i标签差不多，该标签可以指定id，style,class等核心属性，还可以定义onclick等事件属性

strong标签定义粗体文本，与b标签的用法和效果基本相同

small标签定义小号字体文本，常用与标注免责声明，注意事项，法律规定和版权相关的声明字体，该标签可以指定id，style,class等核心属性，还可以定义onclick等事件属性

sup标签定义上标文本， 该标签可以指定id，style,class等核心属性，还可以定义onclick等事件属性

sub标签定义下标文本， 该标签可以指定id，style,class等核心属性，还可以定义onclick等事件属性

bdo标签定义文本显示的方向， 从左到右排列 从右到左排列 该标签可以指定id，style,class等核心属性，还可以定义onclick等事件属性

HTML 的超链接和锚点

a标签表示超链接，该标签可以指定id，style,class等核心属性，还可以定义onclick等事件属性，还有以下三个重要的属性
  • href：
    • 指定超链接所关联的资源
  • target：
    • 指定使用框架集中的哪个框架来装载资源，该属性值可以是_self、_blank、_top、_parent,分别表示自身，新窗口，顶层框架，父框架
  • media：
    • 指定目标URL所引用的媒体类型，默认值是all，只有当指定了href时才有效

HTML 图像相关元素

img定义图像，只能是一个空元素，不能包含任何内容，可以指定id，class，style等属性和绑定onclick等事件属性
以下两个属性是必须指定的
src指定图片的路径，可以是相对路径也可以是绝对路径
alt指定一段文本，作为图片的提示信息
以下两属性是可选的
height指定图片的高度，可以是百分比，也可以是像素值
width指定图片的宽度，可以是百分比，也可以是像素值
与图片相关的两个标签
map用于定义图片映射，可以包含一个或多个area子元素
area用于定义图片映射的内部区域，只能是一个空元素，除了基本核心属性和事件属性外还可以指定以下几个属性
area的其他属性
onfocus，onblur等焦点属性
shape指定内部区域是哪种区域，默认值是rect即矩形，还有circle，ploy(多边形)
coords指定多个坐标值，用于确定区域位置
href指定区域链接的资源
alt指定指定一段文本，作为提示信息
target指定装载资源的框架
media指定媒体类型，默认是all




HTML 表格相关元素

table标签用于定义表格，只能包含0/1个caption（表格标题），0/1个thead（表格头），0/1个tfoot（表格脚），多个tbody。
核心属性以外的属性
cellpadding 指定单元格与单元格边框之间的间距，可以是百分比也可以是像素值
cellspacing 指定单元格之间的间距，可以是百分比也可以是像素值
width 指定表格宽度，可以是百分比也可以是像素值

tr 定义表格行，只能定义td和th两种子元素

td 定义表格列，除了基本属性外还可以定义以下几个属性
  1. colspan 指定单元格跨多少列
  2. rowspan 指定单元格横跨的行数
  3. height 指定单元格的高度
  4. width 指定单元格的宽度

HTML5 文档结构元素

article 元素代表页面上独立、完整的一篇文章，可以是一个帖子，一篇blog，一篇短文，一条完整的回复
article 使用规则
内部使用header定义文章标题
内部使用footer定义文章的脚注
内部使用section使文章内容分成几个段落
内部可以嵌套多个article，比如blog后面有多个回复

nav元素专门用于定义页面上的导航条，包括页面上方的主导航条，侧边的边栏导航条等

aside元素专门定义当前页面或当前文章的附属信息，通常来说推荐使用css渲染成侧边栏
